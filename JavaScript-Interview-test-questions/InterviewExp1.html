2. What is the output of below-console

const arr = [1, 'q', 3];

arr.map((num) => num*2);

console.log(arr);

output

[1,'q',3]

Explanation:

The .map() function does not change the original array arr. It creates a new array, but that new array is not stored in a variable, so it is discarded.
